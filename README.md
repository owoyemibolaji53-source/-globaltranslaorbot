# 🌍 GlobalTranslaorBot

[![Deploy on Railway](https://railway.app/button.svg)](https://railway.app/template/your-template)

A powerful, fast, and reliable Telegram bot for instant language translation. Translate between 100+ languages with auto-detection, history tracking, and more!

## ✨ Features

- 🌐 **100+ Languages** - Translate between any supported languages
- 🤖 **Auto-Detection** - Automatically detects source language
- 📝 **Multiple Input Methods** - Send text, reply to messages, forward messages
- 📊 **Translation History** - Keep track of your translations
- ⚡ **Blazing Fast** - Optimized for instant responses
- 🔒 **Secure** - Environment variables for sensitive data
- 📱 **User-Friendly** - Inline keyboards and easy commands
- 🎯 **Smart Features** - Language detection, favorites, and more

## 🚀 Commands

| Command | Description |
|---------|-------------|
| `/start` | Start the bot and see welcome message |
| `/help` | Show help guide |
| `/languages` | List all supported languages |
| `/detect` | Detect language of text |
| `/translate [text]` | Translate text to default language |
| `/translate [lang] [text]` | Translate text to specific language |
| `/history` | View your translation history |
| `/settings` | Configure bot settings |
| `/setlang [code]` | Set default target language |
| `/clear` | Clear translation history |
